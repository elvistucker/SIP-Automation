Get List of All Countries Where Products Are Offered

This API returns a list of all countries where one or more specific products are offered.

You can use this command to determine the set of products offered in a geography.

The ID of a country is a 3 letter ISO 3166-1 alpha-3 code.

Parameters

None

Response

Return

Reason

Description

400

Bad Request

The request could not be understood by the server due to malformed syntax.

401

Unauthorized Request

The request requires user authentication. If the request includes user authentication, authorization has been refused for the given credentials.

403

Forbidden

The server understood the request, but is refusing to fulfill it.

404

Not Found

The server has not found anything matching the Request-URI. This response is commonly used when the server cannot provide the reason for refusing the request, or when no other reason is available.

405

Method Not Allowed

The method specified in the Request-Line is not allowed for the resource identified by the Request-URI.

500

Internal Server Error

The server encountered an unexpected condition which prevented it from fulfilling the request.

Example

generic_country_array [

country

]

country {

id (string): A country id, its ISO 3166-1 alpha-3 country code ,

isoAlpha2Code (string): The country ISO 3166-1 alpha-2 country code ,

countryName (string): The country name ,

countryCallingCode (string): The telephony calling code of the country ,

characteristics (nested_characteristics4),

products (Array[nested_products]): The list of available products in the country

}

nested_characteristics4 {

isFixedNumbersOffered (boolean): Are fixed line phone numbers offered in this country ,

isMobileNumbersOffered (boolean): Are mobile numbers offered in this country ,

isPayphoneNumbersOffered (boolean): Are payphone numbers offered in this country ,

standardLeadTime (string): The time it minimally takes to order a new phone number in this country

}

nested_products {

name (string): The product name ,

href (string): A complete URL to the related product

}
